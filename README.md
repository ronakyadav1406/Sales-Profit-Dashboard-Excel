# Sales & Profit Dashboard - Microsoft Excel

## Project Overview

This project is an interactive Sales & Profit Dashboard developed in **Microsoft Excel** to analyze business performance across sales, profit, customers, categories, and states. The dashboard transforms raw data into meaningful business insights through dynamic visualizations and reporting.


## Key Business Insights

- **Technology Category Leadership:** Technology generated the **highest sales** of $704,452.15 and the **highest profit** of $121,861.73, making it the strongest-performing category.

- **Regional Sales Performance:** California emerged as the **top-performing state** with total sales of $390,145.54, contributing significantly to overall business revenue.

- **Profit Growth Trend:** Business profitability **peaked in 2023** with a total profit of $81,786.94, highlighting strong financial performance.

- **Customer Profitability Analysis:** Tamara Chand was the **most profitable customer**, generating $8,981.32 in profit.

## Technical Features

- **Interactive Slicers:** Category, Year, and Month filters for dynamic reporting.

- **KPI Dashboard:** Displays Total Sales and Total Profit metrics.

- **Sales Analysis:** Category-wise and state-wise sales performance tracking.

- **Profit Analysis:** Customer profitability and yearly profit trend evaluation.

- **Data Visualization:** Uses bar charts, pie charts, and KPI cards for better insights.

- **Pivot Tables & Pivot Charts:** Utilized for efficient data summarization and analysis.

## Note
This is a macro-enabled Excel dashboard (.xlsm).

After downloading the file from GitHub, Microsoft Excel may block macros because the file is from an internet source.

To enable full functionality:

1. Download the Sales_Profit_Dashboard.xlsm file.
2. Right-click the file and select Properties.
3. If available, check Unblock and click Apply.
4. Open the workbook in Microsoft Excel.
5. Click Enable Content to use all interactive features.

This is a standard Windows security feature for downloaded macro-enabled files.
